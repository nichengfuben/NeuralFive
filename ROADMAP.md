### 架构演进

#### 单体架构 (当前)
```
neuralfive/
├── core/          # 核心游戏逻辑
├── ai/           # AI引擎
└── ui/           # 用户界面
```

#### 模块化架构 (v1.2+)
```
neuralfive/
├── game/         # 游戏逻辑模块
├── ai/           # AI模块
├── ui/           # 界面模块
├── config/       # 配置模块
└── utils/        # 工具模块
```

#### 微服务架构 (v2.5+)
```
services/
├── game-service/     # 游戏服务
├── ai-service/       # AI服务
├── user-service/     # 用户服务
├── match-service/    # 对战服务
└── analytics-service/ # 分析服务
```
