# SmartFive - AI五子棋游戏

## 项目信息
- 项目名称: SmartFive
- 描述: 一个使用Pygame和Numba的AI五子棋游戏
- 编程语言: Python 3.8+

## 常用命令
- 运行游戏: `python -m src.game.main`
- 运行游戏(Windows): `run_game.bat`
- 运行测试: `python run_tests.py`
- 运行测试(Windows): `run_tests.bat`

## 项目结构
- /src - 源代码
  - /ai - AI算法实现
  - /game - 游戏主程序
- /tests - 测试文件
- /docs - 文档
- /assets - 资源文件

## 编码约定
- 使用Python 3.8+
- 遵循PEP 8代码风格
- 使用类型注解
- 模块化设计

## 重要注意事项
- 游戏使用Pygame库实现图形界面
- AI算法使用Numba进行性能优化
- 项目支持通过setup.py或pyproject.toml安装
