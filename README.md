# Gomoku Master

Gomoku Master is an advanced Gomoku (Five in a Row) game implementation featuring a powerful AI opponent. The game is built with Python and Pygame, offering an engaging and challenging experience for players of all skill levels.

## Features

- Play against a sophisticated AI opponent
- Clean and intuitive user interface
- Visual indicators for the last move and valid moves
- Game state management (win detection, draw detection)
- Responsive design with smooth animations

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/your-username/gomoku-master.git
   ```

2. Navigate to the project directory:
   ```
   cd gomoku-master
   ```

3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

Run the game with:
```
python main.py
```

## Game Rules

Gomoku, also known as Five in a Row, is a strategy board game played on a 15x15 grid. Players take turns placing stones on the intersections of the grid, with the goal of getting five stones in a row horizontally, vertically, or diagonally.
